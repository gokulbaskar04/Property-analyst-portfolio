# Property Analytics & Investment Modelling Portfolio
This portfolio presents a collection of advanced financial modelling and property investment analysis projects, demonstrating practical application of valuation techniques, portfolio optimisation, and strategic decision-making frameworks. The work reflects real-world responsibilities of a Property / Investment Analyst, including feasibility analysis, market research, lease modelling, and investment prioritisation using data-driven approaches.

# Projects
# Project 1: Rent Roll & Lease Register Model

Developed a lease-level rent roll model to analyse tenant income, lease structures, and vacancy risk.

Key Work:

Created a structured rent roll including tenant details, lease start/end dates, and rental values
Modelled annual rental income and lease escalation assumptions
Identified lease expiry risks and vacancy exposure

Outcome:

Enabled assessment of income stability and highlighted lease rollover risk within the portfolio
 
# Project 2: Comparable Sales Analysis (Melbourne)

Built a comparable sales model to evaluate property pricing and market positioning.

Key Work:

Collected and structured recent Melbourne property transaction data
Calculated price per sqm and gross rental yield
Analysed property performance relative to market benchmarks

Outcome:

Supported valuation analysis and identified pricing trends across comparable assets

# Project 3: Melbourne Development Feasibility Model
Developed a full financial feasibility model to evaluate property investment viability.

Key Work:

Built multi-year cash flow projections including rental income, operating expenses, and exit value
Performed DCF valuation with NPV, IRR, and ROI calculations
Conducted scenario and sensitivity analysis on key variables (rent growth, vacancy, discount rate)

Outcome:

Identified investment viability thresholds and key drivers of return performance.

# Project 4: Capital Investment Prioritisation Model

Designed a decision framework to evaluate and rank multiple property investment opportunities.

Key Work:

Compared assets based on IRR, NPV, risk exposure, and strategic alignment
Developed weighted scoring model for investment ranking
Created structured output for investment decision-making

Outcome:

Enabled prioritisation of high-performing assets based on financial and strategic criteria

# Project 5: Melbourne Property Investment Model

Built a comprehensive property investment model to assess long-term performance.

Key Work:

Forecasted rental income and operating expenses
Modelled property value growth and exit scenarios
Evaluated investment returns using DCF metrics

Outcome:

Assessed investment feasibility and long-term return potential


# Project 6: Investment Portfolio Optimisation (LP & ILP)

Developed optimisation models using Linear Programming (LP) and Integer Linear Programming (ILP) to maximise portfolio returns under constraints.

Key Work:

Modelled an 8-asset portfolio using real market data
Applied diversification, sector, and risk constraints
Conducted sensitivity analysis and constraint evaluation
Compared LP (fully diversified) vs ILP (selective allocation) strategies

Key Insight:

LP model delivered 1.67% return under strict diversification constraints
ILP model improved return to 9.48% by selecting high-performing assets

Outcome:

Demonstrated trade-off between diversification and return optimisation

# project 7: Financial Modelling & Monte Carlo Simulation

Built simulation-based models to evaluate uncertainty and risk in investment performance.

Key Work:

Applied probabilistic modelling to simulate multiple investment outcomes
Analysed distribution of returns under varying assumptions
Integrated scenario analysis into financial modelling

Outcome:

Provided deeper insight into downside risk and variability of investment returns
